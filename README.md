## 프로젝트 계획 이유
> 고등학교 팀 프로젝트 발표 아이디어로 평소 만들고 싶었던 객체를 인식하여 자동으로 사격하는 포탑을 제작하게 되었습니다.
> 목표 객체를 인식하고 거리를 구하며, 제어부인 Atmega128에 신호를 보내는 파트를 담당하게 되었습니다.
>
> Raspberry Pi 4 Model B 환경에서 YOLOv3-tiny에 기반한 객체 인식 프로그램을 개발하였습니다.
